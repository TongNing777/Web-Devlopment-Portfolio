# Spacing


**Padding** adds space **inside** an element between the content and its border.
```
Example:
.box {
 padding: 20px;
} 
```


**Margin** adds space **outside** an element, creating space between an element and its surroundings.

```
Example:
.box {
 margin: 20px;
} 
```

**Visualization**
```
  ┌──── Margin (outside the element) ─────┐ 
  │    ┌────── Border ──────┐             │
  │    │   ┌─ Padding ─┐    │             │
  │    │   │ Content   │    │             │
  │    │   └───────────┘    │             │
  │    └────────────────────┘             │
  └───────────────────────────────────────┘
```
